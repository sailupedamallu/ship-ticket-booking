About:

This project is about the Ship-Ticket-Reservation-System which is used to view Ship Schedule, search ship, Seat availability, ship timings. We can also enquire about fare of different ships. We can get information about ship between two port. We can book seats online. This provides a safe and secure seat reservation system.

Online ship Information and Reservation

This Website is built for following purpose:-

  1. View ships Schedule
  2. Search Trains
  3. Seat Availability
  4. ship Timings
  5. Fare Enquiry
  6. ships Between Stations
  7. Booking seats online
  8. Login and Logout Security
  9. Password Changes
  10. Payment Gateway
  11. Ticket Booking History
